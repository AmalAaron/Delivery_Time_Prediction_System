# Delivery Time Prediction System

## Project Overview

This project predicts the estimated delivery time of orders using Machine Learning. It analyzes factors such as distance, weather conditions, traffic, vehicle type, and other delivery-related features to estimate the time required for delivery.

This project was developed as part of my internship at Inspire Softech Solutions.

---

## Features

- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Feature engineering
- Machine Learning model training
- Model evaluation
- Delivery time prediction

---

## Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## Dataset

The dataset contains delivery-related information such as:
- Distance
- Weather
- Traffic Conditions
- Vehicle Type
- Delivery Person Information
- Delivery Time

## How to Run

1. Clone this repository.
2. Install the required libraries:

```
pip install -r requirements.txt
```

3. Open the Jupyter Notebook.

4. Run all cells to train the model and make predictions.

---

## Results

The trained machine learning model predicts delivery time based on the provided input features and helps estimate expected delivery duration.
